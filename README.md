# WebShelf 🌐📚

WebShelf is a clean, minimal, and highly organized single-page dashboard designed to aggregate and categorize your favorite websites. Instead of cluttering your browser with endless bookmarks, WebShelf lets you neatly "stack" your links into custom categories for quick, seamless access.

## ✨ Features

- **All-in-One Dashboard:** View all your essential links on a single, elegant page.
- **Smart Categorization:** Group websites by purpose (e.g., Work, Dev Tools, Social, Entertainment).
- **Fully Responsive:** Access your WebShelf perfectly from your desktop, tablet, or smartphone.
- **Quick Search:** Instantly filter through your stacked sites to find exactly what you need.
- **Lightweight & Fast:** Built with performance in mind for instant loading.

### Prerequisites

You will need a modern web browser and a code editor (like VS Code). If you plan to extend this with a backend, ensure you have [Node.js](https://nodejs.org/) installed.
